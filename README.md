# 🛒 E-commerce Simples

Projeto simples de e-commerce, com back-end em Node.js/Express servindo uma loja virtual básica.

## 🚀 Tecnologias

- [Node.js](https://nodejs.org/)
- [Express](https://expressjs.com/)
- HTML, CSS e JavaScript (front-end)

## 📋 Pré-requisitos

- [Node.js](https://nodejs.org/) instalado (recomendado v18+)

## ⚙️ Instalação

```bash
# Clone o repositório
git clone https://github.com/thbbsa/e-commerce-simples.git

# Entre na pasta
cd e-commerce-simples

# Instale as dependências
npm install
```

## ▶️ Como rodar

```bash
node server.js
```

Depois acesse `http://localhost:3000` no navegador (ajuste a porta caso o `server.js` use outra).

## 📁 Estrutura do projeto
├── data/        → dados dos produtos (JSON)
├── imgs/         → imagens usadas no site
├── public/        → arquivos estáticos (HTML, CSS, JS do front-end)
├── server.js        → servidor Express
├── package.json       → dependências do projeto
└── .gitignore

## 📝 Sobre

Projeto feito como exercício de estudo de back-end com Node.js e Express.
